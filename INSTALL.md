# Инструкция по установке и настройке бота

## 1. Требования

- Python 3.11+
- Telegram-аккаунт
- API-токен RoApp
- Telegram-канал (публичный или приватный)

---

## 2. Создание Telegram-бота

1. Откройте Telegram, найдите бота **@BotFather**
2. Отправьте команду `/newbot`
3. Введите имя бота (например: `Sotik Import Bot`)
4. Введите username бота (например: `sotik_import_bot`)
5. BotFather пришлёт **токен** вида `7123456789:AAHxxxxxxxxxxxxxxxxxxxxxxxxxxxxx` — сохраните его

---

## 3. Создание Telegram-канала

Если канал уже есть — переходите к шагу 4.

1. Откройте Telegram
2. Нажмите меню (три полоски / карандаш) -> **New Channel**
3. Задайте название и описание канала
4. Выберите тип:
   - **Public** — у канала будет @username (например `@mysotik`)
   - **Private** — доступ только по ссылке-приглашению

---

## 4. Добавление бота в канал как администратора

1. Откройте канал -> нажмите на название канала вверху
2. Нажмите **Administrators** (Администраторы)
3. Нажмите **Add Administrator** (Добавить администратора)
4. Найдите вашего бота по username (например `@sotik_import_bot`)
5. Включите право **Post Messages** (Публикация сообщений)
6. Остальные права можно отключить
7. Нажмите **Done** / **Save**

---

## 5. Узнать ID канала

### Публичный канал
Если у канала есть username `@mysotik`, используйте его как ID:
```
@mysotik
```

### Приватный канал (числовой ID)

**Способ 1 — через Telegram Web:**
1. Откройте https://web.telegram.org
2. Перейдите в ваш канал
3. Посмотрите адресную строку — там будет число вроде `-1001234567890`
4. Это и есть ID канала (с минусом в начале)

**Способ 2 — через бота @userinfobot:**
1. Перешлите любое сообщение из канала боту @userinfobot
2. Он покажет chat_id канала

---

## 6. Узнать свой Telegram User ID

1. Найдите в Telegram бота **@userinfobot**
2. Отправьте ему любое сообщение
3. Он ответит вашим числовым ID (например `123456789`)

---

## 7. Установка бота

```bash
# Клонируйте или скопируйте проект
cd tg-channel-import

# Создайте виртуальное окружение (рекомендуется)
python3 -m venv .venv
source .venv/bin/activate

# Установите зависимости
pip install -r requirements.txt

# Скопируйте файл настроек
cp .env.example .env
```

---

## 8. Заполнение .env файла

Откройте файл `.env` в текстовом редакторе и заполните:

```
BOT_TOKEN=7123456789:AAHxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
ADMIN_USER_ID=123456789
DATABASE_PATH=data/bot.db
```

- **BOT_TOKEN** — токен от @BotFather (шаг 2)
- **ADMIN_USER_ID** — ваш Telegram User ID (шаг 6)
- **DATABASE_PATH** — путь к базе данных (можно оставить по умолчанию)

---

## 9. Запуск бота

```bash
python -m bot
```

В логах появится сообщение `Bot started: @your_bot_username` — бот запущен.

---

## 10. Первоначальная настройка через бота

Откройте Telegram и напишите боту в личные сообщения:

### Шаг 1: Настройка подключения
Отправьте `/setup` и следуйте инструкциям:
1. Введите API-токен RoApp
2. Введите ID канала (`@username` или `-1001234567890`)
3. Выберите склад (если их несколько)

### Шаг 2: Выбор категорий (опционально)
Отправьте `/categories`:
- Отметьте нужные категории галочками
- Если ничего не выбрано — импортируются все товары

### Шаг 3: Просмотр и выбор товаров
Отправьте `/products`:
- Просмотрите список товаров с пагинацией
- Отметьте галочками товары для публикации
- Нажмите **Опубликовать выбранные** для отправки в канал
- Или используйте **Предпросмотр** для проверки формата

### Шаг 4: Массовый импорт
Отправьте `/import` для публикации всех новых товаров.

### Шаг 5: Автоимпорт (опционально)
Отправьте `/schedule`:
- Выберите интервал (30 мин, 1 час, 2 часа и т.д.)
- Бот будет автоматически проверять новые товары и публиковать их

## 11. Все команды бота

| Команда        | Описание                                        |
|----------------|-------------------------------------------------|
| `/start`       | Приветствие и список команд                     |
| `/setup`       | Настройка API-токена и канала                   |
| `/categories`  | Выбор категорий для фильтрации                  |
| `/products`    | Просмотр и выборочная публикация товаров        |
| `/import`      | Массовый импорт всех новых товаров              |
| `/history`     | История импортов                                |
| `/reset`       | Сбросить историю публикаций (для повторного импорта) |
| `/schedule`    | Настройка автоматического импорта               |
| `/status`      | Текущий статус бота                             |
| `/help`        | Справка                                         |

## 12. Запуск в фоновом режиме (на сервере)

### С помощью systemd (Linux):

Создайте файл `/etc/systemd/system/tg-import-bot.service`:

```ini
[Unit]
Description=TG Channel Import Bot
After=network.target

[Service]
Type=simple
User=your_user
WorkingDirectory=/path/to/tg-channel-import
ExecStart=/path/to/tg-channel-import/.venv/bin/python -m bot
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

```bash
sudo systemctl daemon-reload
sudo systemctl enable tg-import-bot
sudo systemctl start tg-import-bot
```

### С помощью screen/tmux:

```bash
screen -S bot
python -m bot
# Ctrl+A, D — отсоединиться
# screen -r bot — вернуться
```
